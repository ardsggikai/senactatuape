Como instalar webmin

Passo 1 — Instalando o Webmin
Primeiro, atualize o índice de pacotes do seu servidor se você não tiver feito isso recentemente:

sudo apt update

Então você precisa adicionar o repositório Webmin para que você possa instalar e atualizar o Webmin usando seu gerenciador de pacotes. Para que seu sistema confie neste novo repositório, primeiro você baixará a chave PGP do Webmin e depois a converterá para um formato que o apt possa usar para verificar os arquivos:

curl -fsSL https://download.webmin.com/jcameron-key.asc | sudo gpg --dearmor -o /usr/share/keyrings/webmin.gpg

Essa chave baixada é a mesma que foi usada pelo criador do Webmin para assinar o pacote, e você usará essa chave para verificar a autenticidade do pacote. Para converter o arquivo .asc em um arquivo .gpg funcional, é necessário o comando gpg --dearmor.

Em seguida, você adicionará este repositório ao seu arquivo /etc/apt/sources.list, enquanto faz referência ao seu arquivo recém-convertido que você acabou de adquirir na etapa anterior.

Abra o arquivo em seu editor preferido. Aqui, você usará o vim:

sudo vim /etc/apt/sources.list

Em seguida, adicione esta linha ao final do arquivo para adicionar o novo repositório:

/etc/apt/sources.list
. . .
deb [signed-by=/usr/share/keyrings/webmin.gpg] http://download.webmin.com/download/repository sarge contrib

Salve o arquivo e saia do editor. Se você usou o vim para editar, pode sair pressionando Shift+:, x!, depois ENTER.

Em seguida, atualize a lista de pacotes novamente para incluir o repositório Webmin agora confiável:

sudo apt update

Em seguida, instale o Webmin:

sudo apt install webmin

Quando a instalação terminar, você verá a seguinte saída:
