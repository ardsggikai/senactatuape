Backup do Eclipse:
1) Acessar o workspace
2) Copiar a(s) pasta(s) do projeto(s)

Restore no Eclipse:
1) Acessar o workspace
2) Colar a(s) pasta(s) do projeto(s)
3) File -> Import -> General
4) Selecionar a opção "Existing projects into workspace"
5) Select root directory [Browse] -> selecionar o workspace
6) Selecionar o(s) projeto(s)

Obs: Para abrir uma classe no WindowBuilder, selecionar a classe com o botão direito (open with WindowBuilder Editor)

Dicas eclipse:
<ctrl><shift><f> -> Identação(alinhamento) do código

Dica para nomear um variável composta de 2 palavras usar uma letra maiúscula no início da segunda palavra.
Ex: nomeCliente (camel case)
