Ponto De Crochê

O crochê é uma técnica muito antiga que voltou com tudo. Para fazer os pontos do crochê é preciso manipular a linha, de lã ou de barbante, com uma agulha longa de ponta curva – específica para esse tipo de trabalho. O tamanho da agulha varia de acordo com espessura do fio utilizado.


Tutorial de Crochê

Em minhas peças, trabalho com aumentos aleatórios, ou seja, só coloco aumentos quando o crochê ‘pede’, quando vejo que está começando a levantar as bordas. Uma forma mais fácil de descobrir quando é hora de colocar aumento na sua peça é prestar atenção quando o ponto da carreira começa a ficar inclinado para a direita… Quando isto acontece é hora de colocar ponto duplo. Mas, cuidado!!! Não faço muitos se não vira ‘onda’.

Como Fazer Um Tapete

Agora que você já sabe os pontos básicos de crochê é hora de dar um passo adiante e começar a fazer peças de crochê inteirinhas. Os tapetes de crochê simples são uma ótima opção para isso, afinal, você coloca em prática o que aprendeu e ainda fica com uma peça para decorar sua casa ou até mesmo para vender. Aliás, se você gosta de fazer crochê nada melhor do que investir para fazer dessa atividade uma profissão.
